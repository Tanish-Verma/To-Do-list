# To-Do List Bot Road Map

This document keeps track of the current features and the prospective features of the Task Manager telegram bot.

## Current features 

The bot currently supports these features 

- Add a task: Users can add new tasks.
- List tasks: Displays the current list of pending tasks
- Mark task as done: Allows users to mark an individual task as completed.
- Clear the entire task list: Provides an option to reset the task list.
- Multi-User support

## Upcoming features

- Bulk Delete: Introduce a feature to delete multiple tasks simultaneously.
- View Completed Tasks: Allow users to view tasks they have marked as completed.
- Pagination: Implement pagination for the task list and delete options to improve navigation with many tasks.
- Task Priority: Add the ability to set and manage the priority level for tasks
- Edit Tasks: Enable users to modify existing tasks
- Categories: Sort tasks based on categories
- Due Dates: Allow users to assign and track due dates for tasks

The following features are currently being considered for development and will be added to the bot in future updates. Please note that this list is subject to change as development progresses and new possibilities arise.
